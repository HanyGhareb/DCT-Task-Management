The list standard

Every large list (Admin users & audit, PC all petty cash, DT all requests, HR employees) gets this exact pattern: server query with { items, total, limit, offset }, skeleton rows while loading, a shared pager, 300 ms debounced search, and an error path that toasts + offers retry. A simulated server (312 rows, 600 ms latency) powers this page.
