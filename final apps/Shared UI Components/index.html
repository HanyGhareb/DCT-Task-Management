Add a component pattern

  1. Create components/<name>/index.html — a single self-contained, runnable mockup (link Google Fonts, inline CSS/JS, use the platform.css token names so it ports cleanly).
  2. Copy the Drawer Form card above, point href + the preview iframe at your new folder, and set the status badge.
  3. Drop a NOTES.md beside the mockup: what it is, its params / classes, and the promote path into final apps/shared/ (CSS block + component module + the one-line wiring each app needs).
  4. When approved, promote once: styles → shared/css/platform.css, behaviour → shared/js/<name>.js, then bump APP_VERSION in every app per the shared-change rule.
