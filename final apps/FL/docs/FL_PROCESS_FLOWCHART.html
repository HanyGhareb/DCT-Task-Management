Running Alongside the Main Flow

Supporting processes that can fire at any point of an active engagement — each with its own approval template.

↻ AMENDMENT / RENEWAL

Contract Changes

Amendments rebuild the schedule — rebuild_payment_schedule preserves PAID rows and regenerates unpaid periods. Renewals spawn a successor contract and complete the original.

⏰ DAILY SWEEP

Document Expiry Alerts

send_expiry_alerts runs daily — flags documents as EXPIRING_SOON / EXPIRED and can block new contracts or vouchers until IDs and visas are renewed.

⚙ SELF-SERVICE

Profile Change Requests

Freelancers request bank / email / phone changes via the portal; applied by apply_profile_change after FL_ADMIN approval — bank changes never bypass review.
