Overview

This document outlines the full project plan to build the i-Finance Task Management application using Oracle APEX on Oracle Cloud Infrastructure (OCI) Free Tier. APEX provides a complete low-code development platform — UI, authentication, charts, file handling, and REST APIs — all built-in and hosted directly on the Autonomous Database.

Key APEX Advantage vs Oracle JET: Oracle APEX is pre-installed on the Always-Free Autonomous Database. No VM, no Node.js, no build tools, and no ORDS setup required. The entire application — frontend, backend, and database — runs on a single managed service.

Target Architecture

Browser (Oracle APEX UI) ↕ HTTPS (port 443) Oracle APEX Engine ────── Built into Autonomous Database ↕ PL/SQL Packages (Business Logic) ↕ Oracle 23ai Tables / Views ↕ OCI IAM (Identity Domain) ── OAuth2/OIDC for authentication

Project Phases

Phase 1
OCI Infrastructure Setup
Goal: Provision Autonomous DB and access APEX — estimated 0.5 day
Unlike Oracle JET, no VM is required. Oracle APEX and ORDS are pre-installed on the Always-Free Autonomous Database. The APEX URL is provided automatically after provisioning.
#TaskDetail
1.1Create OCI CompartmentIsolate all project resources in a dedicated compartment
1.2Provision Oracle 23ai Autonomous DB (Always Free)Select "Transaction Processing" type — APEX is automatically included
1.3Access APEX AdministrationUse the DB console link: Tools → Oracle APEX → Open APEX
1.4Create APEX WorkspaceWorkspace name: IFINANCE — linked to a new DB schema IFINANCE
1.5Create Workspace Admin AccountSet workspace admin credentials for the development team
1.6Verify APEX URLConfirm public HTTPS access to the APEX URL (format: https://<db>.adb.<region>.oraclecloudapps.com/ords/apex)
1.7Configure custom domain (optional)Use OCI DNS + Load Balancer to map a custom domain to the APEX URL
Phase 2
OCI IAM & APEX Authentication Setup
Goal: Configure identity, roles, and Social Sign-In — estimated 1 day

2.1Create IAM Identity DomainUse Default domain or create a dedicated domain for the application
